Bienvenue sur votre espace perso.

Pour commencer à utiliser votre site

Ce n'est pas très compliqué

Passez la souris pour plus d'informations et commencer à utiliser votre espace



Dans le champ « Hôte » (1), veuillez saisir « sftp://myshop.itinet.fr ».

Dans le champ « Identifiant » (2), veuillez saisir votre identifiant Myshop.

Dans le champ « Mot de passe » (3), veuillez saisir votre mot de passe Myshop.

Dans le champ « Port » (4), veuillez saisir « 3022 ».

Puis cliquez sur « Connexion rapide ».



Il est possible que cette fenêtre apparaisse, cliquez sur « Ok » et cochez éventuellement la case.



Vous retrouvez l’arborescence de votre ordinateur, ainsi que celle de cotre espace personnel sur Myshop. Vous pouvez transférer des fichiers par un simple « cliqué-glissé ». Vous disposez d’un espace d’1 Go.
